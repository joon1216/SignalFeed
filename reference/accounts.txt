# SignalFeed 벤치마크 계정 목록 (자동 발굴 입력)
# 한 줄에 하나씩. '#'로 시작하는 줄은 무시.
#
# 형식:
#   ig:계정명          (Instagram — @ 없이)
#   yt:채널URL         (YouTube — /shorts 탭 URL 권장)
#
# 발굴:  venv/bin/python backend/reference/discover.py            # 후보 선정만
#        venv/bin/python backend/reference/discover.py --collect  # 선별 게시물 수집까지
#
# 인스타 로그인(권장): instaloader --login <내계정> 으로 세션 생성 후
# .env에 IG_SESSION_FILE=세션파일경로 설정. 없으면 익명 시도 (제한 엄격).
#
# 예시 (실제 계정으로 교체):
# ig:economy_account_example
# yt:https://www.youtube.com/@channel_example/shorts
